Mendel 3D

This model is meant to be constructed from scratch including printing the actual components. The controller features a RepRap Arduino Mega Pololu Shield (RAMPS).

The aim of this project is to learn about 3D printers. The wiring and structure of the printer was completed before I received the model. Large modifications to the wiring, axis calibration, and software configuration was needed to start printing. Lots of documentation can be found online for the process of completing the 3D printer.

skills used.

These are the technical skills learned and used to complete the project.

  • Wiring

    Wiring from the arduino controllers to the stepper motors needed design and efficiency planning.

  • Software Confgiuration

    Open source software was needed to convert STL files to G-code for the 3D printer.

  • Calibration

    Both the XYZ axis as well as the extruder needed calibration for accurate printing.

  • Construction

    Learning the architectual design of the printer for reconstructing certain components.

Printer Structure

The structure of the printer consists of several stepper motors, 3D printed components, and straight metal axis. The printer can organized into 3 main parts: the extruder cartridge, the printer bed, and the super structure. All these components need to be calibrated well enough before actual printing.

Wire Management

The RAMPS board is used as the microcontroller to understand g-code and move the stepper motors. Branching off of the board, wires have to be connected to the motors and end stops of the 3 axis. Thermistors and power resistors used for the extruderand heat bed also need to be wired to the microcontroller.
