meisbags

Meisbags is an ecommerce website that focuses on selling hand made bags with personally chosen fabrics.

The aim of this project is to learn about ecommerce and create a website that is able to sell prodcuts. I used many tools to create the website including Wordpress and Stripe. SSL and PCI compliance technology is also implemented to the site to that ensure proper security measures are taken in the checkout process.

skills used.

These are the technical skills learned and used to complete the project.

  • Wordpress

    The Wordpress platform was used for business logic and a content management system.

  • Secure Socket Layer

    SSL integration with custom checkout was done to ensure great site-wide security.

  • Custom CSS

    A base theme was implemented along wiht custom CSS to modify the template for personal use.

  • PCI Compliance

    Stripe was used to process credit card payments and ensure PCI compliance.

Wordpress CMS

The main wbesite platform used to create meisbags was the Wordpress content management system. This provided a easy way to modify website content in a high level form. Many third party plugins were used such as Woocommerce and WP Security. Custom CSS was added to the default theme to further personalize the website to my liking.

Payment Security

Implemented an SSL certificate to ensure secure payments from users. Stripe, a third party payment gateway, was integreated to the store to allow for PCI compliant payments. No sensitive information is ever stored on the website but rather communicated with encryption and private keys.
